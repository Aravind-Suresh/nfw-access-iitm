# nfw-access-iitm
A shortcut python script to automate net-access login

## Requirements

selenium>=2.53.6

## Trial

First clone the repository,

```
$ git clone https://github.com/Aravind-Suresh/nfw-access-iitm.git
$ cd nfw-access-iitm
```

To install dependencies, run

```
$ pip install -r requirements.txt
```

For a sample run,

```
$ python main.py
```
